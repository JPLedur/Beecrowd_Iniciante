## 📚 Sobre o Beecrowd

O [Beecrowd](https://www.beecrowd.com.br/) (antigo URI Online Judge) é uma plataforma brasileira de juiz online que oferece centenas de problemas de programação divididos em categorias como Iniciante, Strings, Estruturas de Dados, Matemática, entre outras.

A categoria **Iniciante** é composta por problemas que envolvem:
- Entrada e saída de dados
- Operações aritméticas básicas
- Estruturas condicionais
- Laços de repetição
- Conceitos fundamentais de programação

## ▶️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/JPLedur/Beecrowd_Iniciante.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd Beecrowd_Iniciante
   ```

3. Compile o arquivo desejado:
   ```bash
   javac src/Problema_1000.java
   ```

4. Execute:
   ```bash
   java -cp src Problema_1000
   ```
