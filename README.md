## 🚀 Como Executar

**Pré-requisito:** Java JDK instalado.

```bash
# Compilar
javac src/NomeDoArquivo.java -d bin/

# Executar
java -cp bin NomeDoArquivo
```

**Exemplo:**
```bash
javac src/HelloWorld.java -d bin/
java -cp bin HelloWorld
```
